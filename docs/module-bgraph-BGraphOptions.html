Class: BGraphOptions

bgraph~ BGraphOptions

new BGraphOptions(options)

bgraph object constructor. Creates an empty beeminder graph and/or road matrix table with the supplied options. Particular goal details may later be loaded with bgraph~loadGoal or loadGoalFromURL functions.
Parameters:
Name Type Description
options BGraphOptions JSON input with various graph options
Properties:
Name Type Description
options.noGraph boolean Generates an empty graph and JSON if true
options.divGraph object Binds the graph to a div element
